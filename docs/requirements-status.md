# WorkLogLite 需求与状态

最后更新：`2026-06-09`

本文档只记录当前仓库对应的真实状态，不保留已经下线或回退的历史结论。

## 产品定位

`WorkLogLite` 当前定位为：

- 原生 `Windows` 离线工作日志工具
- 单机使用
- 可便携分发
- 兼顾日常办公记录和轻量工具工作区

## 当前稳定能力

### 1. 日志主流程

已实现：

- 按日历选择日期
- 查看当天条目列表
- 新增、编辑、保存、删除条目
- 批量删除选中条目
- 条目列表按开始时间、分类、状态排序

### 2. 富文本编辑

已实现：

- `RichEdit` 正文编辑
- 基础格式工具栏
- 编辑区最大化
- `Ctrl + 滚轮` 调整正文显示缩放
- `Ctrl + 0` 恢复默认缩放
- 时间分隔行插入

当前约束：

- 仍然是富文本编辑器，不是实时 Markdown 所见即所得
- Markdown 相关只保留低风险的一键转换能力

### 3. 工作区

已实现：

- 截图
- 常用回复
- 复制时间戳
- 专注计时
- 单位/带宽计算器
- 数据目录/截图目录/日志目录
- 便签
- 取色器
- 纯贴
- 录系统声
- 录麦克风

### 4. 录音

已实现：

- 录系统声与录麦克风
- 保存对话框保留
- 默认文件名按时间生成
- 最长录制 `5` 小时
- 超时自动停止
- 输出 `WAV`

### 5. 数据管理

已实现：

- 导出全部数据
- 导入全部数据
- 清空全部数据
- 自动备份旧数据
- 危险路径拦截

### 6. 报表

已实现：

- Markdown 报表
- 按分类报表
- CSV 导出

## 已明确下线项

- 密码登录/改密功能已取消
- 护眼模式未保留

## 已知剩余风险

这些不是编译级风险，而是更偏人工回归：

- 不同 DPI 下的窗口布局观感
- 录音依赖真实音频设备环境
- 截图、取色等系统能力在目标机器上的交互稳定性
- 少量历史中文字符串仍可能存在编码污染痕迹

## 发版建议

当前版本可以作为正式发布版本。

发版前人工最少回归建议：

1. 新建一条日志，验证保存、排序、删除、批量删除
2. 验证富文本缩放、最大化编辑、时间分隔插入
3. 验证录系统声和录麦克风各一次
4. 验证导出、导入、清空数据各一次
5. 验证便携包解压后直接运行
