# WorkLogLite

`WorkLogLite` 是一个原生 `Win32` 的离线工作日志工具，定位是单机、轻量、可便携分发。

当前版本重点解决的是：
- 按日期记录和查看工作条目
- 富文本编辑、预览和常用格式操作
- 条目级材料路径管理
- 常用办公小工具集成
- 数据导入、导出、清空和恢复

## 当前功能

### 日志与条目
- 左侧日历按天切换
- 右侧条目列表显示当前日期可见条目
- 支持新增、编辑、保存、删除、批量删除
- 列表支持按日期范围、分类、状态排序
- 支持跨日期范围条目显示
- 保存后程序会恢复为单选状态，不再残留异常多选

### 富文本编辑
- 正文使用 `RichEdit`
- 支持加粗、斜体、下划线、项目符号、编号、对齐、缩进
- 支持插入时间分隔行：`----YYYYMMDD----`
- 支持 `Ctrl + 鼠标滚轮` 缩放正文显示
- 支持 `Ctrl + 0` 恢复 `100%`
- 支持编辑区最大化

### 材料路径
- 所有类型的条目都可以设置材料路径
- 设置或清除材料路径后立即持久化，不再依赖后续手工保存
- 打开材料目录失败时会明确提示，不再静默失败
- 如果条目没有单独材料路径，仍保留默认材料目录回退逻辑

### 工作区
集成了常用本地工具：
- 截图
- 常用回复
- 复制时间戳
- 专注计时
- 单位 / 带宽计算器
- 数据目录
- 截图目录
- 录音目录
- 日志目录
- 材料根目录
- 便签
- 取色器
- 纯贴
- 时间分隔
- 录系统声
- 录麦克风

### 录音
- 分别支持系统声和麦克风录音
- 保留保存对话框
- 默认文件名按时间生成
- 最长录音 `5` 小时，超时自动停止
- 输出为兼容性更好的 `WAV`

### 数据管理
- 导出全部数据
- 导入全部数据
- 清空全部数据并自动备份
- 危险路径和错误导入源会被拦截
- 导入 / 清空后如果界面重载失败，会明确提示

### 报表
- Markdown 报表
- 按分类汇总
- CSV 导出

## 数据目录

程序优先使用可执行文件旁边的 `data/` 目录。  
如果当前目录不可写，会回退到：

`%LOCALAPPDATA%\WorkLogLite\data`

主要文件：
- `data/YYYY/MM/YYYY-MM-DD.wlr`
- `data/categories.txt`
- `data/tasks.wlt`
- `data/recurring_meetings.wlrp`

说明：
- 所有数据都保存在本地文本文件里，便于备份和迁移
- 当前版本已取消密码功能，不再有登录或改密流程

## 运行

直接运行：
- `build\WorkLogLite.exe`
- 或便携版 `portable\WorkLogLite.exe`

分发优先使用：
- `WorkLogLite-portable.zip`

## 构建

在 Windows 下执行：

```bat
build.bat
```

输出：

```text
build\WorkLogLite.exe
```
