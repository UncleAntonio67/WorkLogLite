# WorkLogLite

`WorkLogLite` 是一个原生 `Win32` 的离线工作日志工具，定位是单机、轻量、可便携分发。

当前版本重点解决的是：
- 按日期记录和查看工作条目
- 富文本编辑、预览和常用格式操作
- 条目级材料路径管理
- 左下角工作区常用工具
- 本地数据导入、导出、清空和恢复

## 当前功能

### 日志与条目
- 左侧日历按天切换
- 右侧条目列表显示当前日期可见条目
- 支持新增、编辑、保存、删除、批量删除
- 列表支持按开始日期、分类、状态排序
- 支持跨日期范围条目显示
- 保存、重载、删除后程序会恢复为单选状态，不再残留异常多选

### 富文本编辑
- 正文使用 `RichEdit`
- 支持加粗、斜体、下划线、项目符号、编号、对齐、缩进、上下标
- 支持插入时间分隔行：`----YYYYMMDD----`
- 支持 `Ctrl + 鼠标滚轮` 调整正文显示缩放
- 支持 `Ctrl + 0` 恢复 `100%`
- 支持编辑区最大化
- 手动保存和自动保存后会保留当前光标与选区，不再跳回开头

### 材料路径
- 所有类型的条目都可以设置材料路径
- 设置或清除材料路径后立即持久化，不依赖后续手工保存
- 打开材料目录失败时会明确提示
- 条目没有单独材料路径时，仍保留默认材料目录回退逻辑

### 工作区
当前内置工具包括：
- 截图
- 常用回复
- 复制时间戳
- 专注计时
- 单位/耗时计算器
- 数据目录
- 截图目录
- 录音目录
- 日志目录
- 材料根目录
- 便签
- 取色器
- 纯贴
- 时间分隔
- 录系统声
- 录麦克风
- 模板

### 模板
- 模板定义文件：`data/templates.txt`
- 模板按钮不再写入主编辑区
- 选择模板后会直接在 `data/template_drafts/` 下生成草稿文件
- 草稿文件会自动带入模板变量并用记事本打开
- 当前内置示例模板：周报、年报、会议纪要

### 录音
- 分别支持系统声和麦克风录音
- 保留保存对话框
- 默认文件名按时间生成
- 最长录音 `5` 小时，超时自动停止
- 取消保存时会直接丢弃临时录音文件，不再保留也不再打开目录
- 输出为 `WAV`

### 计算器
- 支持单位换算
- 单位前缀当前统一按 `1024` 处理
- 支持“数据量 + 带宽 -> 计算耗时”
- 结果同时输出秒、分钟、小时、天和 `HH:MM:SS`

### 数据管理
- 导出全部数据
- 导入全部数据
- 清空全部数据并自动备份
- 危险路径和错误导入源会被拦截
- 导入/清空后如果界面重载失败，会明确提示

### 报表
- Markdown 报表
- 按分类汇总
- CSV 导出

## 数据目录

程序优先使用可执行文件旁边的 `data/` 目录。  
如果当前目录不可写，会回退到：

`%LOCALAPPDATA%\\WorkLogLite\\data`

主要文件：
- `data/YYYY/MM/YYYY-MM-DD.wlr`
- `data/categories.txt`
- `data/tasks.wlt`
- `data/recurring_meetings.wlrp`
- `data/templates.txt`
- `data/template_drafts/`

说明：
- 所有数据都保存在本地文本文件里，便于备份和迁移
- 当前版本已取消密码功能，不再有登录或改密流程

## 运行

直接运行：
- `build\\WorkLogLite.exe`
- 或便携版 `portable\\WorkLogLite.exe`

分发优先使用：
- `WorkLogLite-portable.zip`

## 构建

在 Windows 下执行：

```bat
build.bat
```

输出：

```text
build\WorkLogLite.exe
```
